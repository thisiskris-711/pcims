# PWA Screenshots

This directory contains screenshots for the PWA install experience.

## Required Screenshots

1. **desktop-wide.png** (1280x720)
   - Desktop dashboard view
   - Shows full interface with sidebar and main content
   - Used for desktop PWA install UI

2. **mobile-narrow.png** (375x667)  
   - Mobile responsive view
   - Shows compact mobile interface
   - Used for mobile PWA install UI

## How to Create Screenshots

### Method 1: Use the Screenshot Creator
1. Open `create-screenshots.html` in your browser
2. Use browser dev tools to set exact dimensions:
   - Desktop: 1280x720
   - Mobile: 375x667
3. Take screenshots of each view
4. Save with the correct filenames

### Method 2: Take from Your Actual App
1. Open your PCIMS dashboard
2. Set browser to 1280x720 for desktop screenshot
3. Set browser to 375x667 for mobile screenshot
4. Take screenshots and save here

## Tools for Screenshot Creation

- **Browser Dev Tools**: Right-click → Inspect → Device emulation
- **Screenshot Tools**: 
  - Windows: Snipping Tool, Lightshot
  - Mac: Cmd+Shift+4, CleanShot X
  - Online: screenshot.net, htmlcsstoimage.com

## Requirements

- Format: PNG
- Exact dimensions required
- Must show actual application interface
- Should demonstrate key features
